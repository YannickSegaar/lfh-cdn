What We’ve Built

We’ve developed a custom AI-powered chat widget for Last Frontier Heliskiing. It answers guest questions about tours, lodges, pricing, safety, and more — using the official LFH knowledge base.

Everything is hosted on a dedicated CDN (GitHub Pages). This means the widget’s code, styling, fonts, and images are all served from one central location. If we need to make any changes — update content, fix a bug, tweak styling — we update the CDN. No WordPress changes needed.

The Embed Code

A single script tag is all that’s needed. It loads the widget, all extensions, and styling automatically.

Add to every page — before </body>
<!-- Last Frontier AI Chat Widget --> <script type="module" src="https://yannicksegaar.github.io/lfh-cdn/loader.js?v=1.1.0" ></script>

WordPress tip: The easiest approach is to add this in the theme’s footer template (footer.php) or via a plugin like Insert Headers and Footers / WPCode. This ensures it loads on every page site-wide.

How the CDN Architecture Works

The WordPress site only has the one script tag. That script pulls everything from the CDN:

Widget engine VoiceFlow chat widget (loaded from VoiceFlow’s CDN)
Styling Custom CSS + Shadow DOM injection for brand colours, fonts, layout
Extensions 9 custom modules — tour explorer, lodge comparison, lead capture, feedback, etc.
Brand assets LFH logo, fonts (Nexa Rust Sans), background images

Any change we make to the CDN is immediately reflected on the website — no WordPress deployment needed.
